# Pan-Tilt Turret

Automatic pan-tilt turret at version 8. Built from scratch — 3D printed housing, DS3218MG servos, ESP32 controller, and a Blender-modeled assembly.

## What it is

A motorized 2-axis platform that can track and aim at targets. Currently used for experimentation with servo control, inverse kinematics, and target detection.

## Hardware

- **DS3218MG** servos (20 kg/cm torque, metal gears)
- **ePETG-CF** housing — rigid, lightweight carbon fiber reinforced filament
- **ESP32** microcontroller (Wi-Fi + Bluetooth)
- Custom 3D printed brackets and mounts

## Status

Version 8 — in progress. Next milestone: target detection via Raspberry Pi Camera.
