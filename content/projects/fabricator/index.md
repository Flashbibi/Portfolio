# Fabricator — Minecraft Server Manager

A desktop GUI tool for managing Minecraft servers on Windows. Built with Flask as the backend and Vue.js for the frontend, packaged into a standalone `.exe` via PyInstaller.

## Why

Managing Minecraft servers manually is tedious — starting/stopping, backups, version switching. Fabricator wraps all of that into one clean interface that runs without installing Python.

## Features

- Start, stop, and monitor servers
- One-click backup and restore
- System tray integration for background operation
- No terminal needed

## Status

Work in progress — core features working, polishing the UI.
