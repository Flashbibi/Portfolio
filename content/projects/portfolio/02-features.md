## Features

- **Terminal Drawer** — Full filesystem navigation, shell commands, guestbook (`sign`, `cat guestbook.md`), easter eggs (`neofetch`, `sudo rm -rf`, `dog`)
- **Boss Fight Easter Egg** — Canvas-based game with wave system, powerups, and a Segfault boss. Hidden entry point.
- **Achievements** — 18 achievements spanning exploration, interaction, and secrets. Progressive disclosure.
- **AI Chat** — Claude Haiku 4.5 via streaming API. Rate-limited per IP via Upstash Redis.
- **GlitchText** — Scheduled word mutations with a four-phase animation (corrupt → morph → settle → exit).
- **Terminal Intro** — Scripted boot sequence on first load: language select, theme select, portfolio launch.
- **Multilingual** — German / English, switchable at any time with a chromatic glitch transition.
- **Dark / Light Mode** — System-default, user-overridable, persisted in localStorage.
- **Destruction Overlay** — `sudo rm -rf --no-preserve-root` triggers a full kernel-panic animation.
- **Cat Mascot** — Long-press draggable. Has opinions.
- **3D Model Viewer** — On project pages with hardware content (e.g. Pan-Tilt Turret).
- **Guestbook** — Real, persistent entries backed by Upstash Redis. Readable from the terminal.
